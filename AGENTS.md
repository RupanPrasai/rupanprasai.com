# Project Guidelines

- Keep pages static and deployable on Cloudflare Pages.
- Preserve canonical domain: `https://rupanprasai.com`.
- Maintain unique SEO metadata (title/description/canonical/OG/Twitter) per page.
- Keep JavaScript minimal; prefer Astro + semantic HTML.
- Update profile/project placeholders in `src/site.config.ts` rather than scattering values.
